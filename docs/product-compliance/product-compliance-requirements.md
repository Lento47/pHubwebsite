# Requisitos de cumplimiento del producto PymeHub

**1. Aceptación legal versionada.** El producto DEBE registrar qué documento aceptó cada usuario, versión, fecha, IP y contexto.

**2. Separación de roles.** Distinguir datos del CLIENTE (workspace) de datos propios de OTNEL S.A. (facturación, soporte).

**3. Controles UI mínimos:**
- Enlace permanente a Términos y Privacidad.
- Pantalla de aceptación al alta.
- Centro de cuenta con datos fiscales.
- Generación de solicitudes de privacidad.

**4. Auditoría de logs.** Registrar: login, cambio de pass, alta/baja user, exportaciones, cambios de rol, borrado de documentos.

**5. IA y Transparencia.** UI clara indicando qué contenido es generado por IA. Los resúmenes son apoyo operativo, no asesoría profesional.
