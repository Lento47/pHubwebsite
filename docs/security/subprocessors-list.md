# Inventario de Subprocesadores (Subprocessors List)

Este documento detalla los terceros autorizados por Otnel S.A. para tratar datos por cuenta del Cliente (Workspaces) en la plataforma PymeHub.

| Nombre | Función | Región de Tratamiento | Fundamento Legal |
| --- | --- | --- | --- |
| **Amazon Web Services (AWS)** | Infraestructura Cloud y S3 | USA / Global | MSA / DPA |
| **OpenAI** | Procesamiento de IA y LLM | USA | Terms of Service (API) |
| **Resend** | Notificaciones por correo | USA | Standard Contractual Clauses |
| **Digital Ocean** | Hosting y Redis | USA / EU | MSA |
| **Stripe** | Procesamiento de pagos | Global | PCI-DSS Compliance |

**Nota:** Otnel S.A. mantiene contratos de privacidad y seguridad con cada uno de estos proveedores para asegurar que el tratamiento de datos se limite estrictamente a las instrucciones de PymeHub.
