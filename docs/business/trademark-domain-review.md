# Revisión de nombre comercial, marca y dominio para PymeHub

**Objetivo.** Antes de invertir en branding, ventas o publicidad, Otnel S.A. debe verificar que el nombre comercial, la marca y el dominio tengan una probabilidad razonable de uso pacífico.

**Proceso de revisión:**
1. Definir nombre principal, variantes, abreviaturas y tagline.
2. Revisar disponibilidad de dominio .cr, .lat, .com y variantes defensivas.
3. Revisar colisiones obvias en el Registro de Propiedad Intelectual de Costa Rica.
4. Revisar presencia comercial y redes sociales.
5. Definir estrategia de solicitud marcaria si el nombre pasa la depuración.

**Reglas internas:**
- Evitar nombres genéricos como “software pymes Costa Rica”.
- Evitar signos demasiado descriptivos.
- Evitar acercamiento fonético innecesario a marcas existentes.
- Registrar el dominio principal en cuanto se apruebe el nombre.
- Resultado esperado: Un memo corto con “apto / apto con riesgo / no apto”, fecha, búsqueda realizada y decisión.
