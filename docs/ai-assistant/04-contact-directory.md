# 04 — Directorio de Contacto Oficial

## Reglas de enrutamiento
1. Nunca dar un contacto sin antes intentar resolver el problema.
2. Dar SIEMPRE el contacto correcto segun el tipo de caso.
3. Nunca exponer ltranav@pymeshub.lat para problemas tecnicos rutinarios.

## Directorio

### Soporte tecnico y operativo
**support@pymeshub.lat**

Usar para:
- Bugs confirmados de la plataforma (P1, P2, P3, P4)
- Problemas persistentes que el asistente no pudo resolver
- Incidentes de servicio caido o perdida de datos
- Errores de facturacion o cobros incorrectos
- Solicitudes de recuperacion de datos

Al escribir, incluir siempre:
- Nombre del workspace afectado
- Plan actual
- Descripcion del problema y cuando empezo
- Mensaje de error exacto (si aplica)

### Ventas y planes comerciales
**sales@pymeshub.lat**

Usar para:
- Informacion sobre plan Enterprise
- Cotizacion personalizada
- Demo de la plataforma
- Contratos, SLA personalizado, acuerdos especiales
- Modalidad Enterprise on-premise
- Distribuidores o revendedores

### Fundador y asuntos estrategicos
**ltranav@pymeshub.lat**
**Leonardo Trana — Co-fundador de PymeHub**

Usar SOLO para:
- Propuestas de alianzas o partnerships
- Asuntos legales o contractuales de alto nivel
- Feedback critico de roadmap
- Incidentes de seguridad graves que requieren atencion ejecutiva

NO usar para soporte tecnico rutinario ni preguntas de uso.

## Respuestas modelo

Bug confirmado:
> Revisamos juntos los puntos principales y el problema parece ser un fallo de la plataforma.
> Te recomiendo escribir a: support@pymeshub.lat
> Incluyeles: nombre del workspace, plan actual, descripcion del problema, cuando empezo y el mensaje de error exacto.

Ventas:
> Para hablar sobre planes, cotizaciones o una demo, puedes escribir a: sales@pymeshub.lat

P1 critico:
> Esto parece un incidente critico. Escribe de inmediato a: support@pymeshub.lat
> Con el asunto: [P1] - [descripcion breve del problema]

Fundador:
> Para asuntos estrategicos puedes contactar al fundador: ltranav@pymeshub.lat
> Si tu consulta es soporte tecnico o planes, te recomiendo support@pymeshub.lat o sales@pymeshub.lat para atencion mas rapida.
