Introducción a PymeHub

Bienvenido a PymeHub, el socio inteligente diseñado para centralizar la operación completa de tu PyME en una sola herramienta.

Propuesta de Valor

"Tu socio inteligente que te dice exactamente qué ajustar en tu negocio, basado en datos."

¿Qué es PymeHub?

PymeHub es una plataforma SaaS B2B **multi-tenant** desarrollada específicamente para el mercado hispanohablante, con especial enfoque en las necesidades operativas de Costa Rica. Centralizamos lo que normalmente requeriría 5 o 6 herramientas distintas (Slack, HubSpot, Trello, Excel) en un único ecosistema fluido.

Funcionalidades de Élite

  • Inbox unificado: Gestiona Email y WhatsApp en una sola vista.
  • CRM Inteligente: Control total de clientes, proveedores y leads.
  • Insights IA: Análisis predictivo mes a mes en español.
  • Facturación Hacienda: Integración nativa para facturación electrónica en CR.
  • Automatizaciones: Reglas sin código para ahorrar horas de trabajo.

Autenticación API

La API de PymeHub usa JWT (JSON Web Tokens) con un sistema de refresh tokens para sesiones seguras de larga duración.

URL Base

https://api.pymeshub.lat

Formato de Respuesta

Todas las respuestas son JSON con la siguiente estructura estándar:

// 200 OK
{
  "data": { ... },
  "message": "OK"
}
// 401 Error
{
  "statusCode": 401,
  "message": "Token inválido",
  "error": "Unauthorized"
}

Endpoints Principales

POST /api/auth/register

Crea una nueva cuenta y workspace de forma programática.

REQUEST BODY: 
{
  "email": "tu@empresa.com",
  "password": "contraseña-segura",
  "name": "Tu Nombre",
  "workspace_name": "Mi Empresa S.A."
}
POST /api/auth/login

Obtiene tokens de acceso iniciales.

Cómo usar el token

Incluye el access_token en el header de todas las peticiones:

Authorization: Bearer {token}

Rotación de Refresh Tokens

Cada vez que renuevas el token, PymeHub emite un nuevo refresh token y el anterior queda inválido automáticamente. Esto protege tu cuenta contra robo de sesiones.
