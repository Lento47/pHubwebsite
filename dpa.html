Acuerdo de Procesamiento de Datos (DPA)

Relación entre PymeHub (Procesador) y el Cliente (Responsable).

1. Objeto y Alcance

Este DPA rige el procesamiento de datos personales que el Cliente carga en PymeHub. Otnel S.A. actúa estrictamente como Procesador de Datos siguiendo las instrucciones del Cliente (Responsable).

2. Obligaciones de Seguridad

PymeHub se compromete a implementar medidas técnicas y organizativas para garantizar la confidencialidad, integridad y disponibilidad de los datos. Esto incluye control de accesos, cifrado de datos y auditorías de seguridad periódicas.

3. Subprocesadores

El Cliente autoriza a PymeHub a utilizar subprocesadores de infraestructura de clase mundial (ej. Cloudflare, AWS) para la prestación del servicio. PymeHub garantiza que estos subprocesadores mantendrán niveles de protección equivalentes.
